CS2 - Snap Tap Banned!

Valve's latest update for CS2 has officially disallowed the use of SnapKey and similar software/hardware solutions that manage keyboard input. These tools, including Snap Tap and others, are now disallowed on official Valve servers. If CS2 detects automated input from these sources, players will be kicked from the server (not banned) but can rejoin once the prohibited software or function is disabled.
